====================
本程序用于运营商号段配置文件生成
====================

1、程序基于python3编写，运行脚本需要安装python3环境（Excel xlsx file; not supported报错是由于xlrd1.2.0之后的版本不支持xlsx格式，
需卸载高版本xlrd）
pip uninstall xlrd
pip install xlrd==1.2.0

2、处理文件需放在新增号段文件夹中，可批量处理多个号段文件

3、执行生成文件前需先执行“号段处理.py”程序，然后即可执行生成文件（生成文件以覆盖形式写入，原文件无需清空）

4、群组表配置需输入当前年份及批次
